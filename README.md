# autoscript
Automation script - records what you do on the desktop, and saves it in a CSV log with a timestamp

## Requirements
* X
* xautomation

## Usage
```
$ autoscript.sh
```

## Tested with
* Firefox
* SciTE
* Thunderbird
* Xfce4-terminal

## Limitations
* Only tested with Slackware so far
* Auto detection of applications detected is hardcoded
